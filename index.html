Estrutura básica

A estrutura básica é o ponto de partida de todo documento HTML e é essencial entender como estruturar corretamente um arquivo HTML para garantir que o navegador interprete e exiba o conteúdo corretamente.

  1. Declaração do tipo de documento: Inicia com `<!DOCTYPE html>`, que informa ao navegador que o documento está usando HTML5.
  2. Tag `<html>`: Envolve todo o conteúdo do documento.
  3. Seção `<head>`: Contém metadados, como o título do documento e links para estilos e scripts.
  4. Seção `<body>`: Contém o conteúdo visível da página, como texto, imagens e links.

Elementos de texto

(Voltar)

Os elementos de texto são fundamentais para estruturar o conteúdo de uma página web. O HTML oferece diversas tags para organizar e formatar o texto de maneira clara e acessível.

  • Títulos: <h1> a <h6>: Usados para definir títulos e subtítulos, onde `h1` é o mais importante e `h6` o menos.
  • Parágrafos: <p>: Usado para definir blocos de texto.
  • Quebras de linha: <br>: Usado para inserir uma quebra de linha sem iniciar um novo parágrafo.
  • Ênfase: <strong> e <em>: Usados para destacar texto, onde `strong` indica ênfase forte e `em` ênfase leve. É importante entender a diferença semântica entre, por exemplo, <strong> e <b>, ambos deixam o texto em negrito, mas o primeiro indica que o texto é importante.
  • Listas: <ul> (não ordenada) e <ol> (ordenada): Usadas para criar listas de itens, com `li` para cada item da lista.
  • Mark: <mark>: Usado para destacar partes do texto, geralmente com um fundo amarelo.
  • Abreviações: <abbr>: Usado para definir abreviações, com o atributo `title` para fornecer a forma completa.
  • Citações: <blockquote>: Usado para citações longas, geralmente com recuo.
  • Citações curtas: <q>: Usado para citações curtas dentro de um parágrafo, geralmente com aspas.
  • Referências: <cite>: Usado para citar o título de uma obra, como livros ou filmes.
  • Subscrito e sobrescrito: <sub> e <sup>: Usados para texto em subscrito (abaixo da linha de base) e sobrescrito (acima da linha de base), respectivamente.

Links e Imagens

(Voltar)

Links e imagens são elementos essenciais para a navegação e a apresentação visual de uma página web. Eles permitem que os usuários acessem outras páginas e visualizem conteúdo multimídia.

  • Links: <a>: Usado para criar links, com o atributo `href` para especificar o destino. Exemplo: `<a href="https://www.exemplo.com">Visite nosso site</a>`.
  • Imagens: <img>: Usado para inserir imagens, com o atributo `src` para especificar o caminho da imagem e `alt` para fornecer uma descrição alternativa. Exemplo: `<img src="imagem.jpg" alt="Descrição da imagem">`.

Tags de Estruturação Semânticas

(Voltar)
  • Header: <header>: Usado para definir o cabeçalho de uma página ou seção, geralmente contendo o título e a navegação principal.
  • Nav: <nav>: Usado para definir uma seção de navegação, contendo links para outras páginas ou seções do site.
  • Main: <main>: Usado para definir o conteúdo principal da página, excluindo cabeçalhos, rodapés e barras laterais.
  • Section: <section>: Usado para definir seções temáticas dentro do conteúdo, cada uma com seu próprio título.
  • Article: <article>: Usado para definir conteúdo independente e autocontido, como postagens de blog ou artigos de notícias.
  • Aside: <aside>: Usado para definir conteúdo relacionado, mas não essencial, como barras laterais ou notas de rodapé.
  • Footer: <footer>: Usado para definir o rodapé de uma página ou seção, geralmente contendo informações de copyright, links de contato e outras informações complementares.
  • Div*: <div>: Usado para agrupar conteúdo genérico, sem significado semântico específico, mas útil para aplicar estilos ou scripts.
  • Span*: <span>: Usado para agrupar texto ou outros elementos inline, sem significado semântico específico, mas útil para aplicar estilos ou scripts.

    • * Não possuem significado semântico associado
